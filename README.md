# 🌍 Global Superstore Analytics Dashboard

## 📊 Overview

This project presents an interactive **Global Superstore Analysis Dashboard** built to analyze sales, profit, and operational performance across different regions, categories, and segments.

The dashboard provides key insights into:

* Sales trends over time
* Profit performance
* Regional distribution
* Shipping cost analysis
* Customer segmentation

---

## 🚀 Key Features

* 📈 **Sales & Profit Analysis**

  * Year-wise trends of sales and profit
* 🌎 **Geographical Insights**

  * Country-level performance visualization
* 🧩 **Segment Analysis**

  * Consumer, Corporate, and Home Office segmentation
* 📦 **Shipping Cost Breakdown**

  * Cost comparison across shipping modes
* 🗂️ **Category Distribution**

  * Sales split across Furniture, Technology, and Office Supplies
* 🔘 **Interactive Filters**

  * Region and Quarter slicers

## 📂 Dataset

* Global Superstore Dataset (commonly used retail dataset)
* Includes:

  * Orders
  * Sales
  * Profit
  * Shipping details
  * Customer segments

---

## 💡 Insights Derived

* Technology category generates the highest revenue
* Standard Class shipping has the highest cost contribution
* Consumer segment dominates overall sales
* Significant growth trend observed from 2011 to 2014

---

## ⚙️ How to Use

1. Download the `.twbx` file from the `dashboard/` folder
2. Open using Tableau Desktop
3. Use filters (Region, Quarter) to explore insights
